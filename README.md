# CLI-Note
Set a note/message for a file on the CLI

## Installation
* Create directory /etc/note/
* Git clone or wget note into /bin/
* chmod +x /bin/note

## Usage

### Creating a note

Syntax

    note set <filename> <message>

Example

    note set nano test

### Reading a note

Syntax

    note read <filename>

Example

    note read nano
